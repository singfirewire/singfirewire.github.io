รวบรวมผลงาน

นี่คือตัวอย่างผลงานการทำเว็บไซต์ด้วย node.js ที่เปิดให้คนทั่วไปใช้งาน:

  • SRT/TXT Merge

    รวมไฟล์ TEXT สำหรับงาน SUBTITLE

  • School Photo Service Calculator

    คำนวณต้นทุนกำไรสำหรับรับงานถ่ายรูปโรงเรียน

  • PINTO

    หน้าเว็บสำหรับจับเวลาคงเหลือ

  • DAENG NAM NGERN

    เทียบราคาสินค้า

  • tax-cal

    คำนวณค่าบริการที่ต้องการรับยอดเต็ม

  • zakat-calc

    คำนวณซากาตประจำปี

  • QR Code Generator

    สร้าง QR Code ได้อย่างง่ายดายและรวดเร็ว โดยไม่มีโฆษณารบกวน เหมาะสำหรับการใช้งานในงานต่างๆ ที่ต้องการสร้าง QR Code

  • URL Shortener

    ย่อ URL ให้สั้นลงได้ทันที โดยไม่มีโฆษณาและใช้งานง่าย เหมาะสำหรับการแชร์ลิงก์ที่ยาวให้สั้นลงและดูเป็นมืออาชีพมากขึ้น

  • Label Generator

    ระบบออกแบบและสร้างใบปะหน้าพัสดุอย่างมืออาชีพ พร้อมระบบติดตามสถานะการจัดส่ง ช่วยให้การจัดการพัสดุเป็นเรื่องง่าย

  • Price Calculator

    คำนวณต้นทุนขาย ช่วยคำนวณราคาขายที่เหมาะสมและจัดการต้นทุนได้อย่างมีประสิทธิภาพ

  • Event Timer

    จับเวลางานอีเว้น ระบบจับเวลาสำหรับการจัดกิจกรรมและงานอีเว้นต์ต่างๆ ใช้งานง่ายและมีประสิทธิภาพ
